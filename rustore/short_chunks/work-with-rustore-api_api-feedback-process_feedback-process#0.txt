Разработчик может получить отзывы на приложение:

с помощью метода «Получение отзывов приложения»;
в формате CSV — метод «Получение отзывов в формате .csv»;



Разработчик может ответить на отзыв, оставленный пользователем — метод «Оставить ответ на отзыв»;


Узнать статус модерации ответа на отзыв — метод «Получение статуса ответа на отзыв»;


Можно изменить уже опубликованный ответ на отзыв — метод «Изменение ответа на отзыв»;


Доступно удаление ответа на отзыв — метод «Удаление ответа на отзыв»;
