Для работы большинства SDK необходимо, чтобы приложение было подписано правильной подписью. Чтобы узнать, какой подписью подписан файл APK, нужно воспользоваться дополнительными инструментами, которые покажут отпечаток подписи.Информация о подписи указана в файле build.gradle. Если не указать подпись явно, используется подпись по умолчанию ~/.android/debug.keystore.Ниже рассмотрены способы, с помощью которых можно узнать подпись.
