В Remote Config можно просмотерть подробные сведения по результатам экспериментов в виде графиков.Откройте Remote Config.
Перейдите на страницу с результатами эксперимента.
Нажмите на пересечение метрики и варианта.
Во всплывающем окне нажмите «Детальные данные по метрике».Откроется страница с детальными результатами эксперимента.
