После завершения скачивания APK-файла обновления вы можете запустить установку обновления. Для запуска установки обновления вызовите метод completeUpdate().try {  await RustoreUpdateClient.completeUpdate();} catch (err) {  console.log(err);}Обновление происходит через нативный инструмент Android. В случае успешного обновления приложение закроется.
