Управление доступами и ролями пользователей в RuStore Консоли


Вы можете установить ограничение возможных операций с приложениями для сотрудников и назначить им определённые роли. Это позволит чётко настроить варианты взаимодействия с приложениями. Например, разрешить сотруднику только публиковать приложения или совершать возвраты.


Как добавить пользователей
​






Выберите приложение.






Перейдите на вкладку «Доступы» в меню слева.






Нажмите кнопку «Пригласить пользователя» в верхней части экрана.












Укажите VK ID пользователя. 
Как узнать VK ID?






Выберите роль пользователя. Можно выбрать только один вариант из списка.










Нажмите кнопку «Пригласить».






Как редактировать доступы
​






Выберите пользователя.






Нажмите на 
 рядом с ним.






Выберите новую роль.










Нажмите кнопку «Сохранить».




Как удалить пользователя
​






Выберите пользователя.






Нажмите на 
 рядом с ним.










Нажмите кнопку «Удалить».




предупреждение
Будьте внимательны, отменить удаление не получится и придётся заново добавлять пользователя в случае ошибочного удаления.


Какие роли можно выбрать
​


В RuStore возможны шесть ролей пользователей:




владелец компании;


администратор;


релиз-менеджер;


разработчик;


финансовый менеджер;


специалист поддержки.
