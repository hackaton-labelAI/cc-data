Для подключения необходимо скачать RuStore Review SDK и импортировать его в проект (Assets > Import Package > Custom Package). Зависимости подключаются автоматически с помощью External Dependency Manager (включён в SDK).Minimum API level должен быть установлен не ниже 24. Минификация приложения (ProGuard/R8) в данный момент не поддерживается, необходимо отключить её в настройках проекта (File > Build Settings > Player Settings > Publishing Settings > Minify).
