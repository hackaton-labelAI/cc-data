Результаты сравнения выбранного варианта в эксперименте с  контрольным вариантом.Доверительный интервалПоставьте галочку, чтобы  отображать «Доверительный интервал» для контрольного или тестового варианта.Доверительный интервал показывает, в каком диапазоне расположатся результаты выбранные изменения метрики.Чем больше значение Alpha, тем шире доверительный интервал.
К примеру, при Alpha 0.01, с вероятностью 99%, изменение метрики находится в указанном доверительном интервале.Доверительный интервал сужается по мере повышения точности эксперимента.
