Создайте новую папку с именем проекта.В папке проекта создайте файл package.json — этот файл используется для указания всех пакетов. необходимых вашему приложению.В папке проекта создайте папку public. В этой папке создайте два новых файла:index.html — этот HTML-файл будет отображать визуализированную карту, с которой ваши пользователи смогут взаимодействовать.
site.css — этот файл будет содержать CSS, необходимый для правильного форматирования карты и боковой панели.
