Запрос передается в формате JSON в HTTP body. JSON запроса состоит из обязательных и не обязательных полей. В URL-адресе запроса должны быть заданы обязательные параметры.Ниже представлен простой пример JSON-запроса.{   "locations":[      {         "lat":43.133200,         "lon":131.911300      },      {         "lat":50.266000,         "lon":127.535600      }   ],   "costing":"auto",   "costing_options":{      "auto":{         "use_border_crossing":0      }   },   "directions_options":{      "units":"kilometers"   },   "id":"my_route"}В этом запросе строится маршрут между Владивостоком и Благовещенском. Он строится так, чтобы избежать маршрута через Китай, добавляя штраф за пересечение международных границ. Полученный маршрут отображается в километрах.
