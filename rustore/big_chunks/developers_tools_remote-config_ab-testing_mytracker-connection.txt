Как подключить MyTracker


Для работы с А/В эскпериментами требуется подключение к MyTracker, иначе некоторые возможности будут недоступны. С помощью MyTracker вы можете добавить метрики и отслеживать их. Метрики позволяют понять, успешен ли ваш эксперимент.


Пример
​


Вам нужно увечилить продажи внутри приложения. У вас есть гипотеза, что пользователи охотнее купят товары, если обновить навигацию в меню. Создайте эксперимент, в рамках которого часть пользователей увидят обновлённое приложение, и добавьте метрику для отслеживания его результатов.


Как подключить MyTracker
​




Откройте 
Remote Config
.


Выберите приложение.


Перейдите во вкладку «A/B эксперименты».


Нажмите «Подключить MyTracker». Подключение происходит в сервисе «Инструменты» в RuStore Консоли.








После подключения MyTracker нажмите «Добавить метрику». (См. 
Метрики
)
