Метод TracerCrashReport.log(Throwable), служащий для отправки non-fatal, объявлен deprecated и будет вскоре удален. Вместо него используйте метод TracerCrashReport.report(Throwable). Это не касается метода TracerCrashReport.log(String), служащего для добавления логов к событиям crash и non-fatal, он продолжит работать без изменений. Актуальные примеры использования на странице «Crash и ANR».Начиная с версии 0.2.3 изменяется способ подключения tracer-plugin. Зависимость заменяется с ru.ok.tracer:plugin:0.1.1 на ru.ok.tracer:tracer-plugin:0.2.3 (обратите внимание, изменена не только версия). Plugin id меняется с ru.ok.tracer.mapping_plugin на ru.ok.tracer. В дальнейшем версии плагина и runtime-зависимостей будут совпадать, публиковаться они будут вместе. Актуальная информация на странице «Быстрый старт».
