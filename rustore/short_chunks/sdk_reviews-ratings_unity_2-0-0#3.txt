Вызовите RequestReviewFlow() заранее, перед вызовом LaunchReviewFlow(), чтобы подготовить необходимую информацию для отображения экрана. Срок жизни ReviewInfo — около пяти минут.RuStoreReviewManager.Instance.RequestReviewFlow(    onFailure: (error) => {         / Handle error     },    onSuccess: () => {         / Handle success     });Если получен ответ onSuccess, то можно в течение примерно пяти минут запустить процесс запроса оценки и отзыва о приложении LaunchReviewFlow.
Если получен ответ onFailure, то не рекомендуется самостоятельно отображать ошибку пользователю, так как пользователь не запускал данный процесс.к сведениюСписок возможных ошибок
