Метод позволяет получить статус модерации ответов на отзывы или получить информацию на отдельно взятый ответ на отзыв.Ограничения:по умолчанию на каждой странице отображается по 20 отзывов. Вы можете отображать до 100 отзывов на странице, указав значение в параметре size;
совместно использовать параметры пагинации и фильтрации по 1 ответу на отзыв — нельзя (либо пагинация с указыванием пары параметров page и size либо id).
