Определите URL-адрес для сервера, который будет получать уведомления.
Сервис по отправке уведомлений работает только с теми адресами, которые начинаются на https://.Ваш сервер должен поддерживать TLS-соединение сертификатом, который выпущен центром сертификации.
Например, Let’s Encrypt.
Подходят в том числе сертификаты, выпущенные Национальным удостоверяющим центром Минцифры РФ.Если ваш сервер развёрнут в публичном облаке, сертификат также можно получить у облачного провайдера.
