Вы можете авторизоваться в RuStore через Сбер ID:




Нажмите 
 на домашнем экране вашего устройства.


Нажмите «Войти».


Нажмите 
, чтобы авторизоваться через приложение Сбербанк.


Войдите в приложение Сбербанка, чтобы авторизоваться в RuStore.






 




Как добавить карты из Сбер ID
​


После авторизации вы можете разрешить привязку карт из Сбер ID в профиль RuStore. Для этого:




Поставьте галочку рядом с пунктом «Добавить к аккаунту ранее сохранённые карты в Сбер ID». Так вы подтвердите своё согласие.


Нажмите кнопку «Продолжить».


Карты из Сбер ID появятся в разделе «Способы оплаты».
