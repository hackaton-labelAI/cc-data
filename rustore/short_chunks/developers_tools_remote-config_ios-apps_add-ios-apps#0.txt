Предварительные условия:для подключения вашего приложения к Remote Config не обязательно иметь аккаунт в RuStore;
добавлять приложения к аккаунту может только его владелец.Откройте Remote Config.
Перейдите на вкладку «Приложения».Нажмите  , чтобы добавить приложение в систему.


Введите имя для приложения и выберите тип платформы «IOS».подсказка Что такое функционал предварительной проверки? При добавлении вы сразу можете включить предварительную проверку при изменении и создании параметров. Это позволит протестировать конфигурацию перед применением, а также, при необходимости, назначить ответственных.При включённой настройке:
 после создания эксперименты переходят в статус Pending и требуют обязательного теста вариантов. Только после теста вариантов эксперимент перейдет в статус Ready to run и будет запущен в указанную дату старта;
 подтверждение параметров и условий после их создания становится обязательным.
